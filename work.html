    Projects

    The largest and most interesting project that I have worked on to date is the greenfield project that currently occupies 90% of my role at my place of work, ASX listed SaaS provider JCurve Solutions. The project is cloud based multi-tenented business software delivered as a service. The software architecture comprises of two primary applications supported by a number of microservices.

    The application I have spent most time on is the API. The API is a seperate application to support whitelabel licensing and is designed to be modern, stateless, RESTful and resource oriented.

    The API is built in Ruby and is nominally running Rails, though certain key aspects of the architecture have been customised. Postgres provides the persistence layer and Redis provides caching. As the product is new, we have the luxury of using edge versions in our stack.

    Since the product is proprietary it wouldn't be appropriate to provide specific details about it's design, but for more detail about the technologies I've encountered both on and off the clock, refer to the skills page.
